# 1. 배경: 복합적 위기의 시대

단순한 선물 거래만으로는 2025년의 극단적 변동성을 방어할 수 없습니다.

### 3가지 핵심 대응 전략
1.  **금융적 전략 (Financial):** 극단적 시나리오를 가정한 자본 잠식 방어.
2.  **운영적 전략 (Operational):** 실물 인도 비용(Basis)과 물류 리스크 관리.
3.  **기술적 전략 (Technical):** AI와 대안 데이터(News, Satellite)를 활용한 선제적 예측.

👉 이 전략들을 구체화한 **3가지 사이드 프로젝트 모델**을 제안합니다.
<!-- slide -->
# 2. 제안 모델 개요 (3 Models)

우리는 각 전략의 장점을 극대화한 세 가지 솔루션을 구상했습니다.

| 모델명 | 핵심 초점 | 주요 가치 |
| :--- | :--- | :--- |
| **Model 1: Stress-Test Lab** | 💰 금융 + 🤖 기술 | "내 포트폴리오는 유가 $100에서 생존 가능한가?" |
| **Model 2: Basis-Navigator** | 🚢 운영 + 📊 데이터 | "가장 저렴하고 안전한 물류 경로는 어디인가?" |
| **Model 3: Sentient-CTRM** | 🧠 AI + 📉 리스크 | "시장의 공포 심리를 숫자로 볼 수 있는가?" |
<!-- slide -->
# 3. Model 1: Stress-Test Lab
> **"극단적 시나리오 기반 자본 리스크 측정기"**

### 문제 의식
*   과거 데이터(Backtesting)만으로는 전례 없는 '블랙 스완' 위기를 예측할 수 없습니다.
*   옵션 전략(Collar, Swap)의 비용 대비 방어 효과를 직관적으로 알기 어렵습니다.

### 솔루션
*   사용자가 직접 설정한 최악의 매크로 시나리오에서 포트폴리오의 생존 여부를 시뮬레이션합니다.
*   **Target:** 자금 관리가 중요한 Corporate Treasury, 리스크 매니저.
<!-- slide -->
# 4. Model 1: 핵심 기능

### 🎛️ 커스텀 시나리오 빌더
*   슬라이더 UI로 변수 조절 (예: 유가 +20%, 환율 -10% 동시 발생 시).
*   **결과 시각화:** 예상 자본 잠식률을 신호등(🟢 안전 / 🟡 주의 / 🔴 위험)으로 경고.

### 🤖 AI 시나리오 추천
*   "지금 시장은 2008년 금융위기 직전과 85% 유사합니다."
*   과거 위기 데이터를 학습한 AI가 현재와 가장 유사한 위험 시나리오를 자동 생성.
<!-- slide -->
# 5. Model 2: Basis-Navigator
> **"베이시스 리스크 & 물류 최적화 봇"**

### 문제 의식
*   스크린 상의 국제 시세(LME)와 내가 실제로 지불하는 도입가(Landed Cost)의 괴리(Basis)가 수익을 갉아먹습니다.
*   물류 대란 시 대체 경로를 찾는 것이 인간의 직관에만 의존합니다.

### 솔루션
*   전 세계 주요 항구의 프리미엄과 물류비를 실시간 추적하여 **최적의 소싱 경로**를 찾아냅니다.
*   **Target:** 원자재 구매팀(Procurement), 공급망 관리자(SCM).
<!-- slide -->
# 6. Model 2: 핵심 기능

### 🚢 실시간 프리미엄 트래커
*   주요 항구/광산별 현물 프리미엄 + 해상 운임(Baltic Index) 크롤링.
*   "지금 칠레 대신 호주에서 수입하면 톤당 $50 절감 가능" 인사이트 제공.

### 🔄 물류 유연성 계산기
*   장기 계약(LTA) 위약금 vs 스팟 구매 비용 비교.
*   위성 데이터로 항구 정체를 감지하여 공급 지연 리스크 조기 경보.
<!-- slide -->
# 7. Model 3: Sentient-CTRM (추천 ⭐)
> **"AI 기반 실시간 감성 리스크 매니저"**

### 문제 의식
*   정량적 데이터(가격, 거래량)는 시장의 '패닉'이나 '광기'를 늦게 반영합니다.
*   뉴스와 소셜 미디어의 뉘앙스를 리스크 모델(VaR)에 반영할 방법이 없습니다.

### 솔루션
*   **비정형 데이터(뉴스, 텍스트)**를 분석하여 시장의 심리적 변곡점을 포착하고, 이를 리스크 한도에 실시간으로 반영합니다.
*   **Target:** 프랍 트레이더, 퀀트 운용역.
<!-- slide -->
# 8. Model 3: 핵심 기능

### 🔥 뉴스 감성 히트맵
*   LLM(Solar)을 활용해 글로벌 뉴스의 긍/부정 뉘앙스 분석.
*   품목별(구리, 원유 등) "공포/탐욕 지수"를 실시간 시각화.

### 📉 Dynamic VaR (동적 리스크 관리)
*   감성 지수가 '극단적 공포' 진입 시, 시스템이 자동으로 **VaR(Value at Risk)** 임계치를 보수적으로 조정.
*   "시장 심리가 불안정하니 포지션 한도를 30% 축소하십시오" 알림 발송.
